When you publish a world, you are locking it down so it can no longer be edited. While it's still possible for the world to change while it is active, the state of the world will not save, and the world will reset once everyone has left. Publishing the world also allows you to specify what features you want the world to have.

To publish the world, check the "Publish World" checkbox. To unpublish, uncheck the box.
The options below will only apply to a world that is published.

Allow Editor Tools - When checked, the normal editor GUI is applied to the world. Users can create and destroy objects as usual.
Single Player - When checked, the world will not allow multiple users at once. Instead, each person visiting the world gets a new copy.
Allow Anonymous Users - When you check this, the system will allow anonymous people to log in and use the world. Users will use a default avatar.
Create Avatar for each user - if you uncheck this, users will not automaticlly have an avatar created.
Use Default Camera - when unchecked, you may specify the ID of a camera object to use instead of the default camera.
